## ğŸ§  AI Integration Details

CampusHire leverages the **Sambanova Cloud API** to interpret student resumes in real-time.

1. **Resume Parsing**: The system extracts raw text from uploaded PDF resumes.
2. **Structural Analysis**: The AI evaluates the resume against job descriptors.
3. **Dynamic Scoring**:
   - **Profile Score**: Overall readiness (0-100%).
   - **Technical Score**: Hard skills proficiency.
   - **Communication Score**: Resume clarity and formatting.
4. **Actionable Feedback**: Generates a strictly structured JSON response that powers the dashboard visualizations and a markdown-based "How to get hired" guide specific to the user's missing skills.
